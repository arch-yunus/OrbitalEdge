# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-04-09
### Added
- Google Test (GTest) integration for core analytics.
- Interactive Terminal UI (`orbital_ui`).
- Autonomous Calibration Service template.
- MIT License and Contributing guidelines.

## [1.1.0] - 2026-04-09
### Added
- Astronomical Event Predictor (Retrogrades, Conjunctions).
- High-performance HTML5 Canvas Sky Map.
- SIMD/NEON optimizations for VSOP87.
- Optimized aligned memory layout for coefficients.

## [1.0.0] - 2026-04-09
### Added
- Web Dashboard (v1) with glassmorphism.
- Hardware Sensor Interface (GPS/IMU).
- Benchmarking tool.
- Modern C++ std::optional integration.

## [0.9.0] - 2026-04-09
### Added
- Multi-planet VSOP87 support.
- ELP-2000 Lunar Engine.
- MQTT/IoT serialization.
- Python Bindings.
- ROS2 Telemetry node.

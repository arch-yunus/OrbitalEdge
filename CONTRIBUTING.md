# Contributing to OrbitalEdge

OrbitalEdge, yüksek standartlı bir astronomik mühendislik projesidir. Katkıda bulunmak için lütfen aşağıdaki kurallara uyun.

## 🛠️ Geliştirme Standartları
- **C++ Standart**: C++17 veya daha yenisi.
- **Kod Stili**: Google C++ Style Guide temel alınmıştır. 
- **Dokümantasyon**: Tüm halka açık sınıflar ve fonksiyonlar Doxygen formatında yorumlanmalıdır.
- **Testler**: Yeni eklenen her özellik için `tests/` dizininde bir birim test dosyası oluşturulmalıdır.

## 🚀 Katkı Süreci
1. Projeyi fork edin.
2. Yeni bir feature branch oluşturun (`git checkout -b feature/yeniozellik`).
3. Değişikliklerinizi commit edin.
4. `ctest` ile tüm testlerin geçtiğini doğrulayın.
5. Bir Pull Request açın.

## ⚠️ Lisans
Katkılarınız MIT lisansı altında lisanslanacaktır.
